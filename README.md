### Centinel server

The Server used to control Centinel nodes in the wild.

### Install and usage
#### Debian
    $ apt-get install python-flask python-passlib python-flask-httpauth python-flask-sqlalchemy
    $ python server.py

#### OSX
    $ pip install flask flask-httpauth flask-sqlalchemy passlib
    $ python server.py
